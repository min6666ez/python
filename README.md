# 数据分析学习平台

一个基于 React + TypeScript + Tailwind CSS 的数据分析在线教育平台，专为商务数据分析与应用专业学生设计。

## 核心功能

- **10个数据分析实战项目**：从入门到专家级
- **浏览器端 Python 环境**：使用 Pyodide 运行 Python 代码
- **交互式学习**：实时代码编辑器和结果展示
- **完整的课程体系**：涵盖数据清洗、特征工程、关联规则、聚类分析等
- **响应式设计**：支持桌面端和移动端

## 技术栈

- React 18 + TypeScript
- Tailwind CSS 3
- Vite 构建工具
- React Router v6
- Pyodide 浏览器端 Python
- Firebase Auth (可选)

## 部署

部署到 Cloudflare Pages：
1. 构建命令：`npm run build`
2. 构建输出目录：`dist`
